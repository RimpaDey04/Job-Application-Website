# Job Application Website 

A simple and responsive **Job Application Website** created using **HTML and CSS**. This project demonstrates front-end development skills by designing a professional job application form and layout suitable for real-world use.

---

##  Features

- Clean and modern UI design
- Responsive layout for different screen sizes
- Structured job application form
- User-friendly input fields
- Proper alignment and spacing using CSS
- Beginner-friendly and easy to understand code
